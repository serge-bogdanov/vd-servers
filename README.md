# VDS Landing Page

Современный адаптивный лендинг, разработанный в рамках практики фронтенд-разработки.

## Используемые технологии

* HTML5
* SCSS (Sass)
* JavaScript (ES6+)

## Возможности проекта

* Адаптивная верстка для различных устройств
* Семантическая структура HTML
* Организованная архитектура SCSS
* Интерактивные элементы на JavaScript
* Кроссбраузерная совместимость

## Запуск проекта

1. Клонируйте репозиторий:

   ```bash
   git clone <ссылка-на-репозиторий>
   ```

2. Перейдите в папку проекта.

3. Запустите проект через локальный сервер разработки.

## Цель проекта

Проект создан для практики навыков веб-разработки: создания адаптивных интерфейсов, работы с SCSS, организации структуры проекта и реализации интерактивности с помощью JavaScript.
